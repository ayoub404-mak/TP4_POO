# Objectifs du TP
L’objectif de ce TP est de permettre à l’étudiant de :
— Comprendre et manipuler les classes abstraites.
— Utiliser les modificateurs statiques (attributs et méthodes statiques).
— Manipuler les interfaces, leurs méthodes abstraites, par défaut et statiques.
— Résoudre les conflits entre méthodes par défaut des interfaces.
